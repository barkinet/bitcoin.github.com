Você precisará ter paciência

A sincronização inicial do programa Bitcoin Core pode demorar muito para ser completada. Você precisa se certificar que possui espaço em disco e banda larga suficientes para armazenar todo o tamanho do block chain. Se você souber como fazer o download de um arquivo torrent, você pode acelerar este processo adicionando bootstrap.dat (uma cópia da versão anterior do block chain) na pasta de dados do Bitcoin Core, antes de inicializar o programa.

Bitcoin Core é um projeto livre e de código aberto, lançado sob a licença MIT.
