Sie müssen geduldig sein

Die anfängliche Synchronisierung von Bitcoin Core kann sehr lange dauern. Sie sollten sicherstellen, dass Sie ausreichend Bandbreite und Speicher für die volle Größe der Blockkette zur Verfügung haben. Falls Sie wissen wie man eine Torrent-Datei herunterlädt, dann können Sie den Prozess beschleunigen, indem Sie bootstrap.dat(eine frühere Kopie der Bockkette) in das Bitcoin Core Verzeichniss speichern, bevor Sie die Software starten.
